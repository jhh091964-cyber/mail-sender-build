郵件發送系統 - 使用說明
========================

運行方式
--------
1. 雙擊 main.exe 啟動程式

首次運行
--------
程式會自動創建以下目錄和文件：
- config/
- data/
- data/apis.json
- data/send_logs/
- logs/
- templates/

基本使用
--------
1. 打開「API 配置」標籤，添加 Resend API
2. 打開「模板管理」標籤，創建或導入郵件模板
3. 打開「發送郵件」標籤：
   - 輸入收件人列表（每行一個）
   - 填寫主題和正文
   - 設置發送參數
   - 點擊「開始」發送

功能說明
--------
- API 配置：支持多個 Resend API，可配置 SSH 隧道和代理
- 模板管理：支持變數替換 {{发件人}} {{收件人}} {{链接}} {{随机字符串}} {{日期}} {{UUID}}
- AI 生成：使用 OpenRouter API 自動生成郵件模板
- 批次發送：支持 URL 輪換、同域延遲、進度追蹤
- 日誌記錄：所有發送記錄保存在 data/send_logs/

注意事項
--------
- 請確保網絡連接正常
- 建議先小規模測試
- 發送日誌會記錄到 data/send_logs/ 目錄
